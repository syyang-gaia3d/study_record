3강. 변수
-- 메모리에 데이터를 저장해서 재활용하는 방법

	(1) 변수
		- 변수 : 데이터(자료)를 임시로 담을 수 있는 메모리 공간
	(2) 변수 선언과 초기화
		- 변수 선언 : [타입]과 [변수명]만 정의
		- 변수 초기화 : 대입연산자(=)를 통해 초기값을 지정
	(3) 메모리 할당과 진법
		- 변수 선언과 함께 메모리 할당
		- ex : int는 4byte(1byte = 8bit)
		- 계산기 기능(* 윈도우에 내장된 계산기는 프로그래머용이 있으며, 진법 계산을 할 수 있다.)
		- 기계는 2진법으로만 인지하기 때문에 진법은 참고로 알아둔다. 특히, C와 같은 언어는 진법 계산이 어느정도 필요
	(4) 변수 데이터 변경
		- 선언된 변수는 대입연산자를 통해 언제든 값을 바꿀 수 있다.
		- 한 번 선언한 변수는 다시 선언할 수 없다.(* 같은 메소드 내에 같은 변수는 존재할 수 없다.)
		- 변수를 쓰는 이유?
			-> 재활용하기 위해.
			-> 계속 사용하는 데이터 값을 호출하기 위해 같은 값을 계속 쓰기보다 선언해 둔 변수만 가져와 쓰는 것이 더 편리함
