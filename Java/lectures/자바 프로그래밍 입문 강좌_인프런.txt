*** 기본적인 강의 자료는 구글 드라이브로 공유된 문서가 존재함

1강. 오리엔테이션 -Java 프로그래밍이란?
	(1) Java 언어의 유래, 특징, JDK, IDE 설치 등에 대한 설명
	- Eclipse 화면 설명(* 대부분의 IDE 툴은 비슷한 UI를 가지고 있다)
		-> Project Explorer : 사용자가 사용하는 프로젝트를 한번에 볼 수 있는 일종의 디렉토리 트리
		-> Outline : 코딩한 파일(기능)들 간의 관계 설명 (자주 확인 안 하므로 닫아도 무관)
		-> Task List : 코딩한 기능들을 한번에 확인할 수 있는 창 (자주 확인 안 하므로 닫아도 무관)
		-> Servers : 각종 서버들을 제어하는 창
	(2) 간단한 코딩 : Hello Java World!
		-> 메인 메소드 = 메인 함수 : 프로그램의 출발점을 지정한 메소드
		   main + ctrl&space 를 통해 자동 타이핑 기능 제공
		-> 콘솔창에 어떤 값을 출력하기 위한 명령어 : system.out.println();
		   sysout + ctrl&space 를 통해 자동 타이핑 기능 제공
		-> 프로그래밍이란? 코딩 소스(개발자 업무 영역) =(컴파일)=> 컴파일러 =(기계어 번역)=> 컴퓨터 기계어 인식
			- 이클립스는 저장과 함께 컴파일을 자동으로 진행함. 즉, 특별히 신경쓸 필요 없음
